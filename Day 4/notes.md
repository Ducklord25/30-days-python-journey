# What i have learnt

### String basics: 
- Every text is a string in python. They are written in between single, double or triple quotes. 
- multiline strings can be defined with triple single/double quotes
- we can concatenate strings with the + operator
- len() function returns the length of the string
- \ is an escape sequence
    - \t = tab space (8 spaces)
    - \n = new line
    - \\ = to write a backslash

### String formatting:
- old style (%):
    - %s - strings
    - %d - int
    - %f - float
    - %.2f - 2 digits after decimal
- new style (use {} in string followed by .format(x,y,z...))
- string interpolation ( use f"string {}" )

### Python String Operations
- Strings are like ordered sequences like lists and tuples.
- lang = 'python' is same as ['p','y','t','h','o','n'] list
- many different string methods mentioned in practice.py
